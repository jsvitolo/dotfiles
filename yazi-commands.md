# Yazi - Comandos e Atalhos

## Navegacao

| Tecla | Descricao |
|-------|-----------|
| `j` / `k` | Mover para baixo / cima |
| `h` / `l` | Ir para diretorio pai / entrar no diretorio |
| `gg` | Ir para o primeiro item |
| `G` | Ir para o ultimo item |
| `{` / `}` | Pular para diretorio anterior / proximo |
| `g h` | Ir para home (~) |
| `g c` | Ir para ~/.config |
| `g d` | Ir para ~/Downloads |
| `g D` | Ir para ~/Documents |
| `g .` | Ir para ~/.dotfiles |
| `~` | Ir para home |
| `-` | Voltar para diretorio anterior |
| `Ctrl+o` | Voltar no historico |
| `Ctrl+i` | Avancar no historico |

## Operacoes de Arquivo

| Tecla | Descricao |
|-------|-----------|
| `Enter` | Abrir arquivo |
| `o` | Abrir arquivo |
| `O` | Abrir arquivo interativamente |
| `y` | Copiar (yank) |
| `Y` | Cancelar yank |
| `x` | Recortar |
| `p` | Colar |
| `P` | Colar (sobrescrever) |
| `d` | Mover para lixeira |
| `D` | Deletar permanentemente |
| `r` | Renomear |
| `Ctrl+n` | Criar arquivo/diretorio |
| `.` | Alternar arquivos ocultos |
| `Ctrl+h` | Alternar arquivos ocultos |

## Selecao

| Tecla | Descricao |
|-------|-----------|
| `Space` | Alternar selecao |
| `v` | Entrar no modo visual |
| `V` | Entrar no modo visual (linha) |
| `Ctrl+a` | Selecionar todos |
| `Ctrl+r` | Deselecionar todos |
| `Esc` | Cancelar selecao |

## Busca e Filtro

| Tecla | Descricao |
|-------|-----------|
| `/` | Buscar |
| `?` | Buscar (reverso) |
| `n` | Proximo resultado |
| `N` | Resultado anterior |
| `f` | Filtrar |
| `F` | Busca fuzzy (jump) |

## Ordenacao

| Tecla | Descricao |
|-------|-----------|
| `,m` | Ordenar por modificacao |
| `,M` | Ordenar por modificacao (reverso) |
| `,c` | Ordenar por criacao |
| `,C` | Ordenar por criacao (reverso) |
| `,e` | Ordenar por extensao |
| `,E` | Ordenar por extensao (reverso) |
| `,a` | Ordenar alfabeticamente |
| `,A` | Ordenar alfabeticamente (reverso) |
| `,n` | Ordenar natural |
| `,N` | Ordenar natural (reverso) |
| `,s` | Ordenar por tamanho |
| `,S` | Ordenar por tamanho (reverso) |
| `,r` | Ordenar aleatoriamente |

## Tabs

| Tecla | Descricao |
|-------|-----------|
| `t` | Nova tab |
| `1-9` | Ir para tab N |
| `[` | Tab anterior |
| `]` | Proxima tab |
| `{` | Mover tab para esquerda |
| `}` | Mover tab para direita |

## Visualizacao

| Tecla | Descricao |
|-------|-----------|
| `w` | Mostrar tarefas |
| `z` | Mostrar/ocultar preview |
| `Z` | Mostrar/ocultar preview (painel) |

## Copiar Caminhos

| Tecla | Descricao |
|-------|-----------|
| `c c` | Copiar caminho completo do arquivo/pasta |
| `c d` | Copiar caminho do diretorio atual |
| `c f` | Copiar nome do arquivo |
| `c n` | Copiar nome sem extensao |

## Shell e Comandos

| Tecla | Descricao |
|-------|-----------|
| `Ctrl+s` | Abrir shell no diretorio atual |
| `:` | Modo de comando |
| `;` | Executar comando shell |
| `!` | Executar comando shell (bloqueante) |

## Ajuda

| Tecla | Descricao |
|-------|-----------|
| `~` | Ajuda |
| `q` | Sair |
| `Q` | Sair sem salvar |

## Modo Input (ao renomear/criar)

| Tecla | Descricao |
|-------|-----------|
| `Ctrl+a` | Ir para inicio da linha |
| `Ctrl+e` | Ir para fim da linha |
| `Ctrl+u` | Limpar linha |
| `Ctrl+w` | Deletar palavra anterior |
| `Esc` | Cancelar |
| `Enter` | Confirmar |
